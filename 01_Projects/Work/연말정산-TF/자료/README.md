# 자료

> 전달받은 자료, 참고 링크, 캡처 설명, 정리 메모를 보관하는 폴더입니다.

## 보관 대상
- 교육 자료
- 회의에서 공유된 링크
- 업무 처리에 참고한 사내 문서 위치
- 캡처나 첨부파일에 대한 설명
- 연말정산 제도, 화면, 절차 관련 참고 메모

## 기록 규칙
- 자료만 던져두지 말고 어떤 업무나 일정에 쓰였는지 적는다.
- 민감한 개인정보가 들어간 원본 자료는 저장 위치와 접근 방법만 기록한다.
- 실제 파일을 저장하지 못하는 경우에도 출처, 받은 날짜, 관련 업무 ID를 남긴다.
