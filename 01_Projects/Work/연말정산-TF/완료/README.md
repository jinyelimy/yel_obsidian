# 완료

> 끝난 업무, 오래된 정리 문서, 더 이상 매일 볼 필요 없는 기록을 보관하는 폴더입니다.

## 옮기는 기준
- 업무가 끝났고 [[업무관리]]에 종료 날짜와 처리 내용이 남아 있다.
- 일정이 끝났고 [[일정관리]]에 결과와 후속 액션이 남아 있다.
- 회의록이나 인수인계 문서의 미완료 체크박스가 모두 처리되었다.
- 자주 볼 필요는 없지만 나중에 근거로 찾아야 할 수 있다.

## 옮기기 전 확인
- [ ] 원래 업무 ID 또는 일정과 연결되어 있는가
- [ ] 종료 날짜가 있는가
- [ ] 처리 결과가 요약되어 있는가
- [ ] 남은 액션이 없는가
- [ ] 자료 위치나 산출물 링크가 남아 있는가

## 보관 규칙
- 파일명은 가능하면 유지한다.
- 나중에 날짜순으로 찾기 어렵다면 앞에 `YYYY-MM-DD_`를 붙인다.
- 완료 폴더로 옮겨도 [[업무관리]]의 한 줄 기록은 지우지 않는다.
